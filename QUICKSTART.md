# AdForge — Quick Start

## 3 commands to run

```bash
# 1. Install everything
npm run install:all

# 2. Start the app
npm run dev

# 3. Open in browser
# → http://localhost:5173
```

That's it. Create your first ad by clicking **✦ Create New Ad**.

---

## Requirements
- Node.js 18+ → https://nodejs.org
- npm 9+

## Ports used
- `5173` — React frontend (Vite)
- `3001` — Express backend

## Your data lives here
```
adforge/server/data/ads.json
```
